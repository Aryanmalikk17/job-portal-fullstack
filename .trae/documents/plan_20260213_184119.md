Ah, I see! You are stuck in the `>` prompt because you typed `<<EOF` but didn't finish the command block. The shell is waiting for you to type the SQL commands and then type `EOF` to close it.

### **How to Unstick Yourself**

1. **Type this into the terminal (where you see the** **`>`):**

   ```sql
   INSERT INTO users_type (user_type_id, user_type_name) VALUES (1, 'Recruiter');
   INSERT INTO users_type (user_type_id, user_type_name) VALUES (2, 'Job Seeker');
   EOF
   ```

   *(Make sure to type* *`EOF`* *at the end and hit Enter)*

2. **OR, simpler alternative (Ctrl+C to cancel first):**
   Press **Ctrl+C** to cancel the stuck command.
   Then run this **single-line command** instead (easier to copy-paste):

   ```bash
   docker exec -i jobportal-mysql-prod mysql -u jobportal -pZpluseAppPass2025! -D jobportal_db -e "INSERT INTO users_type (user_type_id, user_type_name) VALUES (1, 'Recruiter'), (2, 'Job Seeker');"
   ```

Once you run that, verify with:

```bash
docker exec -it jobportal-mysql-prod mysql -u jobportal -pZpluseAppPass2025! -D jobportal -e "SELECT * FROM Job Seeker;"
```
