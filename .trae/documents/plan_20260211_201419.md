### Plan: Initialize Git and Pull Code

Since you are already on the server terminal (`root@jobportal-production`), please run the following commands to initialize Git, connect it to the remote, and pull the latest code:

**1. Initialize Git and Set Remote**
```bash
cd /opt/jobportal
git init
git remote add origin https://github.com/Aryanmalikk17/job-portal-fullstack.git
```

**2. Fetch and Reset to Main**
This will force your local directory to match the latest code on GitHub, overwriting any conflicting local files (which is what we want for a deployment).
```bash
git fetch origin
git reset --hard origin/main
```

**3. Rebuild and Deploy**
Now that the code is updated, rebuild the container:
```bash
docker-compose -f docker-compose.prod.yml up -d --build app
```

**4. Verify**
```bash
curl http://localhost:8080/actuator/health
```
